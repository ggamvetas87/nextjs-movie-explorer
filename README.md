# Next.js Movie Explorer

A modern movie search web app built with Next.js.

## Features

- Movie search
- Debounced queries
- Infinite scroll
- Loading skeletons
- Favorites with localStorage
- Movie details page

## Tech Stack

- Next.js
- React
- Tailwind
- External API

## Architecture

Client-side search with debouncing and infinite scrolling to optimize API usage.

## Run locally

```bash
yarn install
yarn dev
```
